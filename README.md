# Mini-Shell-
Un shell bash simple que j'ai codé en C pour Linux 
C'est mon premier projet en C
Ce shell est encore un prototype rudimentaire en cours de développement 



Modifications à venir : 
-Ajout des pipes 
-Ajout des redirection
-AJout des commandes multiples
-Ajout des opérateurs logiques
-Ajout des variables d'environnement
-Ajout de l'autocomplétion 
-Ajout de la commande history 
-Ajout d'un fond d'écran 



1. après installation compiler le script avec la commande suivante : 

gcc minishell.c -o minishell 

2. puis lancer le programme avec la commande suivante 

./minishell
